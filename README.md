# 🚗 Uber Trips Analysis Dashboard

## 📌 Project Overview

This dashboard provides an insightful analysis of Uber trips, focusing on trip duration, distance, purpose, and trip categorization (Business vs Personal).

---

## 🎯 Objectives

* Understand trip patterns and durations by purpose
* Analyze trip distances by purpose
* Explore distribution of trips by business and personal categories

---

## 📊 Key Metrics

* 🛣️ Average Trip Distance: 21.12 miles
* ⏱️ Average Trip Time: 23.24 minutes
* 🚕 Total Number of Trips: 1156

---

## 🔍 Dashboard Highlights

* Meetings and customer visits represent the longest trips by duration
* Business trips make up over 93% of total trips
* Majority of long trips (>60 mins) occur between key locations like Cary and Winston Salem
